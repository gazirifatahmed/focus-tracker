### 🌟 About The App
**Focus Tracker** is an ultra-premium, beautifully crafted self-discipline and habit-tracking mobile application. Built using Flutter and Firebase, it empowers users to break free from bad habits (including NoFap, quitting smoking, screen addiction) and cultivate unbreakable routines. It is **100% free, forever, and completely ad-free.**

## ⚡ Key Highlights

*   **🎮 Gamified Progress:** Track your recovery and habits down to the exact second. Every moment counts toward self-mastery.
*   **🏆 9 Unlockable Meme-Tier Badges:** Earn community-favorite ranks as your streak builds:
    `Clown` ➡️ `Noob` ➡️ `Novice` ➡️ `Average` ➡️ `Advanced` ➡️ `Sigma` ➡️ `Chad` ➡️ `Absolute Chad` ➡️ `Giga Chad`.
*   **📅 Trigger & Setback Logging:** Log every failure with specific dates, metrics, and short, constructive reasons (e.g., *Stress, Tired, Sickness*).
*   **🔔 Interactive Push Notifications:** Supports instant local notifications, congratulatory alerts on level-ups, and dynamic app-sound effects.
*   **☁️ Secure Cloud Sync:** Instantly syncs progress and logs with Firebase Firestore, keeping your history safe across devices.
*   **🌙 Eye-Friendly UI:** Designed with pure dark aesthetics and minimal modern cards for deep focus and self-discipline.

---

## 🛠️ Modern Tech Stack

*   **Client Core:** Flutter & Dart (Compatible with SDK `>=3.0.0 <4.0.0`)
*   **State Management:** Provider Architectural Pattern
*   **Backend Services:** Firebase Auth, Firebase Cloud Firestore, Firebase Cloud Messaging
*   **Media & Sound:** AudioPlayers (custom sound milestones)
*   **Local Engine:** Shared Preferences, TZ (Timezone Offset Mapping), Local Notifications
