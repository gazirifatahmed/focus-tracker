
# 🎯 Focus Tracker

> **Break bad habits. Build streaks. Earn badges. Stay motivated daily.**

Focus Tracker is a premium, beautifully crafted self-discipline and habit-tracking mobile application. Built with Flutter and Firebase, it helps you break bad habits (NoFap, quitting smoking, screen addiction) and build unbreakable daily routines. 100% free, forever, and completely ad-free.

## 🔥 Key Features

* **🎮 Gamified Journey:** Track streaks down to the second in real-time.
* **🏆 9 Unlockable Badges:** Climb from `Clown` ➡️ `Noob` ➡️ `Novice` ➡️ `Average` ➡️ `Advanced` ➡️ `Sigma` ➡️ `Chad` ➡️ `Absolute Chad` ➡️ `Giga Chad`.
* **📅 Setback Journaling:** Log failures with custom tags (e.g., *Stress, Tired, Sickness*) to track triggers.
* **🔔 Smart Push Notifications:** Real-time reminders, custom milestone sounds, and congratulatory alerts.
* **☁️ Firebase Cloud Sync:** Safe, secure, and instant real-time data backup.
* **🌙 Dark & Light Mode:** Eye-friendly, ultra-minimal aesthetic designed for deep focus.

## 🛠️ Tech Stack

* **Frontend:** Flutter & Dart
* **State Management:** `provider`
* **Backend & Sync:** Firebase Auth, Firestore, Cloud Messaging
* **Notifications & Local Storage:** `flutter_local_notifications`, `shared_preferences`
