A habit tracker built with **Flutter** & **Firebase**. Track streaks, earn badges, stay motivated.

---

## ✨ Features

- 🔐 Email/Password Login & Forgot Password
- ⏱️ Real-time timer (HH:MM:SS) with streak counter
- 🏆 9 Achievement Badges (Clown → Giga Chad)
- 📊 Relapse history with delete option
- 🎨 Dark/Light Mode toggle
- 💬 Dynamic motivational quotes
- 🔔 Inspirational push notifications
- ⚙️ Settings: Change Password, Rate Us, Contact Developer, Share App
- 📱 Android, iOS & Web support
